# Booksrus 📖

Booksrus is a comprehensive and modern BFF Graphql-based API to interact with inventory, reserve and purchase books at Books-R-Us!

## Installation 🚀

Use the package manager [yarn](https://yarnpkg.com/) to install dependencies.

```bash
yarn install
```

## Usage 🎡

```cli
yarn start
```
